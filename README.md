# Spotify Customer Churn Dashboard

## Project Overview

This project presents an interactive Spotify Customer Churn Dashboard developed using Microsoft Power BI.

The dashboard helps analyze customer churn patterns by allowing users to filter data and dynamically switch between different customer attributes such as age group, gender, country, subscription type, device type, skip rate, and listening time.

---

## Objectives

- Analyze customer churn behavior
- Identify factors contributing to customer churn
- Provide interactive insights using Power BI
- Build a professional business dashboard

---

## Dataset

The dataset used for this project was obtained from **Kaggle**.

It contains customer demographic information, subscription details, listening behavior, device usage, and churn status, enabling analysis of the factors that influence customer retention and churn.

**Dataset Source:** *(https://www.kaggle.com/datasets/nabihazahid/spotify-dataset-for-churn-analysis)*

---

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Field Parameters

---

## Key Performance Indicators (KPIs)

- Total Customers
- Total Churned Customers
- Churn Rate
- Average Listening Hours

---

## Dashboard Features

- Interactive slicers (Country)
- Dynamic field parameter analysis
- Customer segmentation by age group, gender, country, subscription type, device type, skip rate, and listening time
- Churn analysis across multiple customer attributes
- Interactive KPI cards
- Responsive visualizations with cross-filtering
